# YTLite-ObjC

Pure Objective-C port of [dayanch96's YTLite](https://github.com/dayanch96/YTLite) - A flexible enhancer for YouTube on iOS.

##  Features

-  Remove all ads
-  Background playback
-  Picture-in-Picture
-  OLED dark mode
-  Player gestures
-  Video downloads
-  SponsorBlock integration
-  Return YouTube Dislikes
- ...and 60+ more features!

##  Building
```bash
./Scripts/build.sh
```

Output: `YTLite.dylib`
